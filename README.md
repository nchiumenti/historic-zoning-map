Website repo for Appendix A.2 of [*Under the (Neighbor)Hood: Understanding Interactions Among Zoning Regulations*](https://papers.ssrn.com/sol3/papers.cfm?abstract_id=4082457).

Some source files are not available through this repo because they are too large, 
including the boundary files and raster geotiffs. These are can be found in our
replication package, available through the [REStat Dataverse](https://dataverse.harvard.edu/dataset.xhtml?persistentId=doi:10.7910/DVN/LPSITL).
